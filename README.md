# 移动云盘批量上传工具

这是一个 Windows 桌面小工具，用于把同一批本地文件或文件夹上传到移动云盘多个共享群目标文件夹中。

## 启动

双击 `启动工具.bat`，或在当前目录运行：

```powershell
npm start
```

## 使用流程

1. 在右侧移动云盘页面手动登录账号。
2. 进入“共享群”，打开某个群，勾选要上传到的目标文件夹。
3. 在左侧点击“将选中文件夹加入目标”。
4. 可以继续进入其他共享群，重复添加多个目标文件夹。
5. 在左侧选择本地文件或文件夹。
6. 点击“开始上传”，确认本地内容和目标列表后执行上传。
7. 上传完成后点击“导出 Excel 清单”。

## 安全规则

- 工具不保存账号、密码、短信验证码或 Cookie。
- 只允许把目标加入共享群内的文件夹。
- 上传前必须确认本地内容和目标文件夹。
- 同名文件或文件夹默认跳过，不覆盖云端已有内容。
- 本地选择文件就上传文件，选择文件夹就上传该文件夹及其内部结构。

## 当前限制

- 移动云盘网页结构如果变化，可能需要调整页面适配器。
- 单个文件默认限制为 80 MB，避免通过页面脚本传输超大文件导致卡死。
- 文件夹上传依赖网页中的“新建文件夹”和上传控件可被识别。
